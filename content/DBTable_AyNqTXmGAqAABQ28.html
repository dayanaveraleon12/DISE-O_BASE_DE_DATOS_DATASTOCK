Columns Summary
Column Name Data Type PK / FK Nullable Description
id int(11) PK No

PK Remision, se utiliza cuando una tienda le solicita productos a bodega y bodega le envia/acepta remitir productos a una tienda

id_producto_sitio int(255) FK No

FK Del producto y su sitio actual (solo se pueden solicitar y enviar productos a un y de un solo sitio, siendo Bodega a Tienda ó viceversa)

id_lote int(255) FK No

FK lote, se trae el mismo lote que fue asignado desde el ingreso para cada producto

id_codigo int(11) FK No

FK Codigo, un codigo puede estar ligado a muchos productos (para remitir varios productos en una sola remision)

id_sitio_destino int(255) FK No

FK Sitio, se indica el sitio destino al cual se van a remitir/mover los productos

cantidad_remitida int(255) No

Se indica la cantidad a remitir de cada producto
